# The Seasons Font Files

This directory should contain "The Seasons" font files.

## Required Font Files

To use "The Seasons" font, you need to add the following font files to this directory:

- `TheSeasons-Regular.woff2` (weight: 400, normal)
- `TheSeasons-Bold.woff2` (weight: 700, normal)

## Where to Get The Seasons Font

"The Seasons" is a premium font available from:
- [My Creative Land](https://mycreativeland.com/family/the-seasons/)
- [Fontspring](https://www.fontspring.com/fonts/my-creative-land/the-seasons)

## Font File Format

The font files should be in WOFF2 format for optimal web performance. If you have the font in other formats (TTF, OTF), you can convert them using online tools like:
- [CloudConvert](https://cloudconvert.com/)
- [FontSquirrel Webfont Generator](https://www.fontsquirrel.com/tools/webfont-generator)

## Current Setup

The font is configured in `app/layout.tsx` with Playfair Display as a fallback font. Once you add the font files, the titles will automatically use "The Seasons" font.
